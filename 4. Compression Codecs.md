# Compression Codecs

Recommendations:

| Compression Codec | Best Use Case                                               | What It Does                                                  | Example                                      | Use Case Example                                         |
| ----------------- | ----------------------------------------------------------- | ------------------------------------------------------------- | -------------------------------------------- | -------------------------------------------------------- |
| **LZ4**           | General analytics, real-time queries                        | Fast block compression with very quick decompression          | `column UInt64 CODEC(LZ4)`                   | Integer counters, IDs, general numeric columns           |
| **LZ4HC**         | Data written once, read many times                          | LZ4 with higher compression ratio, slower compression         | `column UInt64 CODEC(LZ4HC(3))`              | Large numeric columns, historical logs                   |
| **ZSTD**          | Large text or JSON, storage‑efficient analytics             | High-compression ratio with configurable levels, moderate CPU | `column String CODEC(ZSTD(1))`               | Strings, JSON, descriptions, messages                    |
| **NONE**          | Extremely latency-sensitive or already compressed data      | No compression                                                | `column UInt64 CODEC(NONE)`                  | Already compressed blobs, small latency-critical metrics |
| **Delta**         | Monotonically increasing integers or timestamps             | Stores difference between consecutive values                  | `timestamp DateTime CODEC(Delta, LZ4)`       | Timestamps, sequential IDs                               |
| **DoubleDelta**   | Time-series timestamps, counters with consistent increments | Stores difference of differences                              | `timestamp DateTime CODEC(DoubleDelta, LZ4)` | Time-series timestamps, sensor readings                  |
| **Gorilla**       | Slowly changing floats, metrics, financial prices           | XOR compression between consecutive floating values           | `price Float64 CODEC(Gorilla, LZ4)`          | Float metrics, stock prices, IoT measurements            |
| **T64**           | Integers with small range or counters                       | Removes unused high bits and packs integers efficiently       | `user_id UInt64 CODEC(T64, ZSTD)`            | User IDs, small-range integer counters                   |
